## Project Overview

This Exploratory Data Analysis (EDA) project investigates hotel booking cancellation patterns using Python, with a focus on identifying the factors associated with reservation cancellations.

The analysis was conducted using Pandas for data manipulation, Matplotlib and Seaborn for data visualization, and includes the exploration of customer behaviour, booking characteristics, seasonal trends, and market segments.

For detailed methodology, visualizations, insights, and business recommendations, please refer to the Jupyter Notebook included in this repository.

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
